# Zomato Restaurant Revenue Predictio
# Project Overview
This project focuses on predicting restaurant revenue using Zomato data.
The aim is to analyze restaurant details and understand what factors affect revenue.

##  Dataset
The dataset is taken from Zomato and includes information such as:
- Restaurant name and type
- City and area (location)
- Ratings
- Cost for two people
- Other related details

Files used:
- `zomato.csv`
- `zomato_cleaned_data.csv`

##  Data Cleaning
- Removed missing values
- Removed duplicate records
- Cleaned and prepared data for analysis

##  Analysis & Visualization
- Exploratory Data Analysis (EDA)
- Data visualized using charts
- Dashboard created using Power BI

Dashboard file:
- `zomato_dashboard.pbix`

##  Model / Prediction
- Analyzed patterns related to restaurant revenue
- Step-by-step explanation provided in the notebook

Notebook file:
- `Restaurant_Revenue_prediction.ipynb`

##  Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas, NumPy, Matplotlib
- Power BI
- Git & GitHub

## Conclusion
This project helps to understand restaurant revenue trends and provides
useful insights for better business decisions.
